# Changelog

Agent Layer publishes full release notes and artifacts on GitHub Releases.

## Where to read release notes

- Releases and binaries: https://github.com/conn-castle/agent-layer/releases
- Canonical changelog file: https://github.com/conn-castle/agent-layer/blob/main/CHANGELOG.md

## What to expect in each release

Each release includes:

- a tagged version with notes
- prebuilt binaries for supported platforms
- `checksums.txt` for verification

:::note
If you pin versions in a repo, read the release notes before updating `.agent-layer/al.version`.
:::

If you are looking for a specific change, search the changelog by version tag (for example `v0.5.0`).
